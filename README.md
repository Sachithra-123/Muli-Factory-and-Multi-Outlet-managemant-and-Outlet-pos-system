Project Description for GitHub
Project Title: DSI Factory Management & POS System
Overview: A robust, enterprise-grade management solution built with Spring Boot and Thymeleaf. This system enables centralized control over multiple factories, stock distribution, and retail outlet sales. It features a premium "SaaS-style" dashboard, real-time inventory tracking, and a dedicated Point of Sale (POS) interface for high-speed transactions.

🚀 Key Features
📊 Dynamic Business Dashboard: Real-time visualization of revenue trends, stock distribution, and low-stock alerts using Chart.js.
📦 Multi-Factory Inventory Control: Track product quantities across different manufacturing locations with automated low-stock thresholds.
🔄 Stock Transfer System: Seamlessly manage the movement of goods from factories to retail outlets with full history logs.
🛒 Advanced POS Interface: A responsive, search-optimized Point of Sale system for cashiers with instant subtotal calculations and automated stock deduction.
👥 Employee & HR Management: Centralized worker records with integrated system access control (Roles: Admin, Cashier).
🧾 Automated Invoicing: Generates professional, print-ready digital invoices for every transaction.
🔐 Enterprise Security: Role-based access control (RBAC) powered by Spring Security to protect sensitive financial and inventory data.
🛠️ Tech Stack
Backend: Java 17, Spring Boot 3, Spring Data JPA, Spring Security.
Frontend: Thymeleaf (Server-side rendering), Vanilla JavaScript (ES6+), Bootstrap 5 (CSS).
Data Visualization: Chart.js for interactive analytics.
Database: MySQL (Relational Database Management).
Icons: Bootstrap Icons.
